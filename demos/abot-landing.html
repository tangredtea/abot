多 Agent 架构

创建任意数量的 Agent，每个拥有独立人设文档（IDENTITY / SOUL / RULES / AGENT），通过路由表自动匹配最合适的 Agent 响应。

🛠️

18 种内置工具

文件读写、Shell 执行、网络搜索、网页抓取、长期记忆、子 Agent 调用、定时任务、技能市场 — 开箱即用，按需启停。

🧠

向量长期记忆

基于 Qdrant 向量数据库，支持语义搜索、分类标签、MMR 重排序。Agent 会记住重要信息，跨会话持续积累知识。

📡

全渠道接入

Web Console、Telegram、Discord、企业微信、飞书 — 一个 Agent 同时接入多个渠道，统一管理消息和会话。

👥

群组协作

创建群聊，拉入多个 Agent。发送消息后系统自动识别意图，分发给最合适的 Agent 回复，支持协作接力和异步委派。

🔐

企业级安全

JWT 认证、RBAC 权限、命令沙箱、路径白名单、资源限额、WebSocket Origin 校验 — 为生产环境设计。
